# ContentLane Design System

Version 1.0

---

# Philosophy

ContentLane is not a dashboard.

It is a premium creative workspace.

Users should feel like they're using:

- Linear
- Raycast
- Framer
- Apple
- Arc Browser

rather than

- HubSpot
- Monday
- Jira
- Generic AI SaaS templates

Every screen should feel calm.

Large whitespace.

Minimal UI.

Strong typography.

One primary action.

---

# Design Keywords

Elegant

Premium

Editorial

Minimal

Calm

Intentional

Confident

Creator-first

---

# Visual Identity

Everything should feel like a product Apple would build if they made AI creative software.

Avoid:

- bright gradients
- neon colors
- glowing borders
- heavy shadows
- glass everywhere
- dashboard clutter

Exactly five reels.

Instagram proportions.

9:16

220-240px wide

390-430px tall

---

Create an invisible circular arc.

Like Apple's Cover Flow.

# Copywriting

Never say

AI-powered

Revolutionary

Next-generation

Best-in-class

Instead

Use concrete language.

Example

"We analyzed your website."

"We found 12 high-converting hooks."

"We generated 8 videos."

---

# Component Rules

Every component must share

same spacing

same radius

same shadows

same typography

No exceptions.

---

# Empty States

Always elegant.

Simple illustration.

One sentence.

One CTA.

Never giant empty boxes.

---

# Loading

Skeletons.

Progressive reveal.

Never spinning loaders in the middle.

---

# Accessibility

Minimum contrast AA.

Buttons

48px+

Inputs

60px+

Focus rings visible.

---

# Responsive

Desktop first.

Tablet

Stack cards.

Mobile

One column.

Keep whitespace.

Never shrink typography excessively.

---

# Final Feeling

Every page should make the user think

"This feels expensive."

Not

"This looks like another AI dashboard."

If a design choice adds visual noise, remove it.

If two solutions exist, always choose the simpler one.
